# Power-Electronics-Modelling-of-the-BMW-eDrive35-Gran-Coupe

System-level modelling and simulation of the BMW i4 eDrive35 electric drivetrain, including vehicle energy analysis, traction inverter design, and power electronics performance evaluation.

## Overview

This project was completed as a team effort and divided into three technical phases:

1. Vehicle Energy Modelling
2. Traction Inverter Design
3. On-Board Charger Design

My primary responsibility was the development and completion of Phases 1 and 2.

## My Contributions

### Phase 1 – Vehicle Energy Modelling

- Vehicle parameter extraction
- Longitudinal vehicle modelling
- Drive-cycle analysis
- Energy consumption calculations
- MPGe analysis
- Well-to-wheel emissions analysis
- Regional electricity generation impact assessment
- Documentation

### Phase 2 – 210 kW SiC Traction Inverter Design

- Three-phase Voltage Source Inverter (VSI) modelling
- Space Vector PWM (SVPWM) analysis
- SiC MOSFET selection and evaluation
- Conduction loss calculations
- Switching loss calculations
- Thermal analysis
- Efficiency optimisation
- MATLAB/Simulink validation
- Documentation

### Phase 3 – On-Board Charger

- Contributed to technical reviews and project integration activities

## Key Results

| Parameter | Value |
|------------|---------|
| Vehicle | BMW i4 eDrive35 |
| DC Bus Voltage | 400 V |
| Rated Power | 210 kW |
| Motor Type | PMSM |
| Inverter Topology | Three-Phase VSI |
| Switching Devices | SiC MOSFETs |
